# C-small_projects
# Proiectele mele în C 🚀

Bine ai venit! Aici adun toate aplicațiile pe care le creez în timp ce învăț limbajul C.

## 🛠️ Programe incluse:
1. **Convertor de temperatură**: Transformă din Celsius în Fahrenheit și Kelvin.
2. **Calculator de economii**: Calculează profitul net dintr-o dobândă bancară.
3. **Disemvowler**: Elimină vocalele din orice text (filtru pentru troli).

## 📚 Ce am învățat până acum:
- Alocare dinamică de memorie (`malloc` & `free`).
- Lucrul cu pointeri și string-uri.
- Formatarea afișării în consolă.
